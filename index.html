My Projects

Album Collection

Album Collection is an app that allows users to view albums with their respective song titles, as well as add and store new song titles for each album. It was developed using ASP.Net MVC Web Application utilizing Entity Framework and Code First migrations to connect to a database. The TDD was developed by testing the controllers with mock repositories using NSubstitute.

Boat Reviews

Boat Reviews is an MVC Web Application that utilizes SQL database to allow users to view reviews on types of boats, as well as create, update, and delete their own reviews. This app was developed by a team who practiced agile methodology, utilized test driven design, and made use of Git branching with GitHub access.
